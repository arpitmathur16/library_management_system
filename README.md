# 📚 Library Management System

A command-line based Library Management System built in Python using Object-Oriented Programming and SQLite for persistent data storage. This project automates core library operations — book inventory, member registration, book issuing/returning, and automatic late fee calculation.

## 🚀 Features

- **Book Management** — Add, search, and view books in the library inventory
- **Member Management** — Register and view library members
- **Issue Books** — Issue books to members with automatic due date assignment (14-day loan period)
- **Return Books** — Return books with automatic late fee calculation (₹5/day if overdue)
- **Overdue Report** — View all currently overdue books with member details
- **Persistent Storage** — All data stored in a local SQLite database

## 🛠️ Tech Stack

- **Language:** Python 3
- **Database:** SQLite3
- **Design Pattern:** Object-Oriented Programming (OOP)
- **Interface:** Command-Line Interface (CLI)
